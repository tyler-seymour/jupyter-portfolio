Network Graph Analysis Notebook ("FleaBay")

OSINT workflow for investigating eCommerce fraud networks. Maps buyers to sellers using publicly available feedback and reviews, allowing investigators to track fake review networks and money laundering paths between usernames. > - NetworkX for constructing graph network and calculating shortest distance between nodes; > - Pandas for scraping and cleaning transaction reviews; > - Matplotlib for static node and edge visualization; > - pyVis for interactive network graph visualization.
